# Admin Form Link/File Upgrade

Added a reference-style admin form layout and new project fields:

- Live Project URL
- GitHub / Source URL
- Documentation Download Path / URL
- Extra Project File Download URL

For GitHub Pages, place PDFs/DOCX/ZIP files inside the `/docs` folder and paste paths such as `docs/q-learning-grid-world-agent-assignment.pdf` into the admin field. A local browser upload option is also available for demo/local use, but committed files inside `/docs` are the clean production method.

The Q-Learning assignment PDF has been added to `/docs/q-learning-grid-world-agent-assignment.pdf` and connected to the Q-Learning portfolio item.
